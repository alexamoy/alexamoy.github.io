Pintrips

A progressive web application that allows users to create, journal, collaborate, and share their trips on 'place boards' in real-time. Pintrips uses a React binding of MapboxGl for each map view, Google's Cloud Firestore to store and sync data, and service workers for offline functionality. Go ahead and start planning your next trip!

An interval training mobile application created using React Native and Google's Cloud Firestore and built in only four days for a hackathon.
